CF1923B题解

发表于 2024-03-01 更新于 2024-06-07 分类于 Solution

思路

很容易证明这是道贪心，贪心策略是先打离自己近的。

证明（不严谨）：

有 222 个怪物 iii 和 jjj，如果 ∣xi∣<∣xj∣\left | x_i \right| < \left | x_j \right |∣xi​∣<∣xj​∣，且先打 jjj 更优。

由于先打 jjj 更优，所以我们在先打 jjj 的情况下自己不会被怪物吃掉。

但是，不管怎样顺序，打死这两只怪物的总时间相等，所以我们先打 iii 和先打 jjj 是一样的。

而且在有些情况下，先打远的会导致近的偷家，于是就没有先打近的优。

CF1922B题解

发表于 2024-01-21 更新于 2024-06-07 分类于 Solution

思路

在不考虑能否构成三角形的情况下，三角形有这几种情况：

  1. 三条边的长度各不相等；
  2. 等腰，其中腰比底长；
  3. 等腰，其中底比腰长；
  4. 等边三角形。

题目中木棍的长度都是 222 的幂，我们再对上述几种情况作进一步分析：

  1. 无法构成三角形。为了构成三角形，我们要让较短的 222 条木棍尽量长，则三个木棍的长度为 2x,2x+1,2x+22^x,2^{x+1},2^{x+2}2x,2x+1,2x+2，明显构不成三角形。
  2. 很明显可以构成三角形。
  3. 无法构成三角形。两条腰长度之和最大也只能等于底。
  4. 很明显可以构成三角形。
